便利機能 VIM7 標準

H, M, L    => 画面をスクロールさせずにカーソルを移動。High, Middle, Low
zt, zz, zb => 現在のカーソル位置を画面上のどこに持ってくるか？t(top),z(middle),b(bottom)
gf         => カーソル下のファイルを開く
C-w f      => Window を split してカーソル下のファイルを開く
K          => カーソル下の word を Manで引く。
netrw      => ファイルエクスプローラ。慣れれば Shellで移動するより格段に楽。

netrw

u, U => 戻る, 進む
v    => 新たな window を開く
o    => open
-    => Parent Directory へ

Mode ( モードにはどんなものがあるか? )

Nomal            => default
Insert           => i,I,a,A,o,O
Visual           => v,C-v
Command          => :
Operator Pending => 操作対象(motion )待ちモード。例えば dj, dw, d$ の時のdの入力後にmotion 待ちの状態

Normal

^   => first word
0   => line head
$   => End of line
w,W   => move forward by word/WORD word-not-skip hiphen
b,B   => same as w/W but backword direction
e,E   => move to end of by word/WORD
yy  => copy current line to register
p   => paste current register-contents
>>  => indent
u   => undo
C-r => redo
/   => search
.   => repead last command
[I  => show line containing keyword under cursor
]I  => same as [I but start searching from current line to EOF
n%  => n%の行に移動。ex) 50%
gg  => first line of file
G   => last  line of file
nG  => n line of file
:n  => n line of file(command mode version)
C-i => Forward
C-o => BackFord
C-a => insert previously inserted text

Visual

v   => 範囲選択(文字指向) visual mode start
o   => move cursor opposit side of selection
O   => move opposit side of selection in 矩形選択モード
V   => 範囲選択(行指向)
C-v => 矩形選択。一括 insert/delete/change
>   => indent
U   => upcase
u   => downcase
iw  => select inner word
aw  => select around(?) word
i{  => select inner of {
a{  => select inner of {
i(  => select inner of (
a(  => select inner of (

Insert

C-p,C-n => 単語補完
C-x C-f => ファイル名補完

Command

r! cat /etc/hosts  => 'cat /etc/hosts'の実行(!)結果(=標準出力)をカーソル下に読み込む(r)
<,>!sort           => 選択範囲をsortコマンドで処理した結果に置き換え
w                  => 保存
w filename         => filename として保存
%s!abc!ABC!gc      => ファイル全体('%')を対象にして、abc > ABCへ変換。一行に何度でも(g),、置き換え前に確認(c)。
set nowrap         => 行の折り返しを無効化
set ft=ruby        => filetype をrubyに設定
set [no]ignorecase => 検索時に大文字小文字を区別しない。
set nowrap         => 行の折り返しを無効化
set [no]hlsearch   => 検索語をハイライトする。
C-f                => 実行予定のコマンドを通常の vim バッファで編集して<CR>で実行
shell              => shell を実行。Exit すると vim に戻る。

Multiple Window

:split               => 水平(horizontal) split
:split file          => 水平(horizontal) splitでfileを開く
:vsplit              => 垂直(vertical) split
:vsplit file         => 垂直(vertical) split でfileを開く
:windo %s!abc!ABC!gc => 置き換えを、全Windowで行う。
C-w s                => horizontal split
C-w v                => vertical split
C-w h,j,k,l          => move cursor to <char> direction window
C-w H,J,K,L          => 指定した方向でWindowを最大化
C-w r,R              => Rotate window
C-w x                => eXchange window
C-w t,b              => select Top/Bottom window

folding ( 折り畳み )

set foldenable
set foldmethod=indent

zf     => create fold (you need :set fdm=manual to this work)
zc, zo => close/open folding
zC, zO => close/open ALL folding under selection
zj, zk => fold 単位で j, k の方向へ移動
set fdc=3 => use left 3 column as forld level indicator

text object

[ 操作 > 範囲 > 対象 ] の組み合わせ。
   d      i      w

diw => Delete Inner Word(word)
diW => Delete Inner WORD(WORD>
ciw => Change ...
ciW => Change ...
yiw => Yank ..
di( => Delete Inner of '('
dib => 上と同じ。
ci[ => Change Inner of '['
daw => Delete A Word(word)
caw => Change A word(word)
ci" => Change Inner '"'
cit => Change Inner Tag(HTML tag)

矩形選択

C-v で矩形選択モードに入る。
矩形選択モードで,Iで挿入すると、複数行に文字を挿入できる。C(change)も同じ。
$ で行末に移動したあと、a で文字を挿入すると、複数行の行末に文字を挿入出来る。
